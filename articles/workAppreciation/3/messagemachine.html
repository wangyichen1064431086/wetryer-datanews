【数据新闻赏析】Message Machine

By 数据传播工作室 阿琛Bonnie

如果你经常关注国际政治新闻，那么你或许常常会惊叹于政治选举运动中各党各派层出不穷的拉票手段。 不过，如果你以为政客们的手段仅止于乐此不疲的广告轰炸、针锋相对的选举辩论，那么就有点out啦~ 据说，美国的政治竞选团队已经悄无声息地把大数据技术应用到这场战争中了！ 想要了解他们是怎么做的吗？——请看本期【数据新闻赏析】为你带来的数据新闻作品 Message Machine 。

Message Machine由美国非营利性调查新闻编辑室ProPublica制作，其诞生与2012年美国总统大选有着密不可分的关系。 当时，一些人收到了来自奥马巴竞选团队的邮件，并对此在Twitter上展开了讨论。 ProPublic公司的Jeff Larson等人很快注意到此事，于是就请ProPublic的读者们把收到的邮件转发给他们。 基于这些邮件，ProPublic公司制作了数据新闻Message Machine ，讲述了一个当时正在兴起的现象—— 政治选举运动中竞选团队运用大数据将特定信息定位给特定人群。

下面就一起来欣赏该作品吧：

Message Machine：

政治选举运动中竞选团队的信息发布方式和经费获取手段真是越来越高明了——他们已经在使用潜在选民们巨大的个人信息数据库了。 这里展示了奥巴马竞选团队发给人们的6个版本的邮件。 这些邮件是全国各地的读者们转发给我们的。 任意点击一个标签即可查看其中一封邮件，然后将鼠标悬停在另一个邮件标签上即可查看悬停邮件和选中邮件的异同。 在选中各个邮件时，邮件内容界面的左侧会显示出相应邮件接收人情况的统计分析图表。
